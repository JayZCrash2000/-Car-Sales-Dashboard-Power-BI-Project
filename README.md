# 📊 Car Sales Dashboard | Power BI Project

## 📌 Overview
This project presents an interactive **Car Sales Dashboard** built using Power BI to analyze sales performance across different dimensions such as time, region, company, and product attributes.

The dashboard provides actionable insights into sales trends, pricing, and performance metrics to support data-driven decision-making.

---

## 🎯 Objectives
- Analyze Year-To-Date (YTD) sales performance  
- Track Monthly-To-Date (MTD) metrics  
- Identify top-performing regions, companies, and car categories  
- Monitor pricing trends and sales distribution  

---

## 📁 Dataset
- Source: CarDekho dataset  
- File: `CAR DETAILS FROM CAR DEKHO.csv`  

---
